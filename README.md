### OBJETIVO DO PROJETO
Nosso projeto consiste em uma estação de monitoramento da qualidade do ar de baixo custo, que utiliza sensores para obter informações climáticas do ambinete, como temperatura e umidade, e captar a concentração de gases poluentes no ambiente, para que, dessa forma, o sistema seja capaz de determinar a qualidade atmosférica do ambiente. Inicialmente, nosso projeto foi desenvolvido para operar em ambientes fechados, mas como é um projeto recorrente e de baixo custo, pode ser facilmente aprimorado, apenas substituindo os sensores para operar em maior escala.

---

### COMPONENTES UTILIZADOS
 
- Sensor MQ-2: Dispositivo eletrônico projetado para detectar a presença de gases inflamáveis e fumaça no ambiente;
- Sensor MQ-7: Sensor responsável por detectar e monitorar a concentração de gás monóxido de carbono (CO);
- Módulo sensor DHT11: Serve para medir a temperatura e a umidade relativa do ar em ambientes;
- Display LCD 16x2: Módulo de exibição alfanumérico de cristal líquido capaz de mostrar 16 caracteres por linha em 2 linhas, totalizando 32 caracteres simultâneos.
- Protoboard: Ferramenta básica amplamente utilizada em projetos eletrônicos para montagem temporária de circuitos, permitindo realizar as ligações elétricas sem a necessidade de soldagem;
- Jumpers: Fios de ligação que servem como condutores elétricos e são essenciais para interconectar os pontos de uma protoboard, permitindo a circulação de corrente entre componentes que não compartilham a mesma coluna de contato.
